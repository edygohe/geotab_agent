**********************************************************************************************
Rutina diaria será:

Escribir código y hacer cambios.
Ejecutar git add . para preparar los cambios.
Ejecutar git commit -m "Un mensaje que describe mis cambios" para guardarlos localmente.
Ejecutar git push para subirlos a GitHub.

**********************************************************************************************


# Antes de subir checar status
git status
------------------------------------------------------------------
# Opción A: Añadir todos los archivos modificados y nuevos
git add .
------------------------------------------------------------------
# Opción B: Añadir solo un archivo específico
git add src/geotab_agent/main.py
------------------------------------------------------------------
# Comentario del cambio
git commit -m "Agregado mensaje de bienvenida al agente principal"
------------------------------------------------------------------
# Ejecutar git push para subirlos a GitHub.
git push

3. Remove-Item -Recurse -Force .git
4. Ejecutar setup_repo.ps1

*********************************
# Instala mermaid-cli si no lo tienes: npm install -g @mermaid-js/mermaid-cli

# Uso: mmdc -i <archivo_entrada.md> -o <archivo_salida.png>

# Ejemplo 1: Convertir el diagrama de la arquitectura general (desde la raíz del proyecto)
mmdc -i ./arquitectura_general.md -o ./docs/diagrams/images/arquitectura_general.png

# Ejemplo 2: Convertir el diseño del orquestador (desde la raíz del proyecto)
mmdc -i ./docs/diagrams/md/orchestrator_design.md -o ./docs/diagrams/images/orchestrator_design.png
